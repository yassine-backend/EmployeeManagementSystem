# Employee Management System

A Python console application for managing employee records.

## Features

- Add employees
- View all employees
- Search employees
- Update employee salary
- Delete employees
- Data validation
- JSON file storage
- Persistent data between sessions

## Technologies Used

- Python
- JSON
- Dictionaries
- Nested Dictionaries
- Functions
- File Handling

## Example Employee Record

```python
{
    "JOHN": {
        "age": 25,
        "salary": 3500,
        "department": "IT"
    }
}
```

## Learning Goals

This project was built to practice:

- Python fundamentals
- CRUD operations
- Functions
- Dictionaries and nested dictionaries
- JSON persistence
- Input validation
- Program structure

## How to Run

1. Clone the repository
2. Open the project folder
3. Run:

```bash
python app.py
```
